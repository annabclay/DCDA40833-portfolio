Hometown Map

Overview

This lab features an interactive Folium map showcasing ten meaningful locations from my hometown of Lexington, Kentucky. Using Python, Folium, and a custom Mapbox style, I geocoded addresses from a CSV file and built an interactive web map with color-coded markers, popups, and tooltips.

Process

To create this map, I started by compiling a list of ten personally meaningful locations in Lexington into a CSV file, including each place's name, address, type, description, and an image URL. I then wrote a Python script using Folium and the Mapbox Geocoding API to convert each address into latitude/longitude coordinates. Each location was assigned a unique marker color and icon based on its category (Restaurant, Cultural, Park, Historical, Recreation, Neighborhood, or Local Business). I also designed a custom Mapbox basemap style to give the map a personalized look and feel.

The script generates styled popups for each marker that display the location's name, type, and a short personal description. A legend was added to the bottom-left corner of the map so viewers can easily identify each category. The final output is a self-contained HTML file that can be viewed in any browser.

Tools Used: Python, Folium, Pandas, Mapbox Studio, Mapbox Geocoding API

Reflection

Creating this interactive map of Lexington was one of the most personally meaningful assignments I have completed in this course. Unlike previous labs that focused on analyzing external datasets or critiquing existing visualizations, this project asked me to turn inward and think about the places that have shaped who I am. Selecting ten locations forced me to be intentional about what matters most to me — from Great Bagel, where I grab breakfast with friends, to Keeneland, a place that connects me to Lexington's deep horse-racing tradition.

On the technical side, this lab pushed me to integrate several tools I had not used together before. Writing a Python script that reads a CSV, geocodes addresses through the Mapbox API, and generates a full interactive HTML map with Folium felt like a real end-to-end data workflow. I especially appreciated learning how geocoding works — the idea that a street address can be transformed into precise coordinates through an API call was fascinating. Debugging the script taught me to pay close attention to data formatting; small issues in the CSV (like extra spaces or inconsistent type names) could break the color and icon mapping.

Designing a custom Mapbox basemap was another highlight. Mapbox Studio gave me control over the visual tone of the map, and I chose a style that felt clean and modern while still being easy to read. It was rewarding to see my custom tiles appear underneath the Folium markers and know that every layer of the map reflected a choice I made.

If I were to expand this project, I would love to add more interactive features — perhaps a sidebar that filters locations by category, or a route that connects all ten spots in a walking tour. I would also want to include actual photographs instead of website links in the popups. Overall, this lab strengthened my understanding of geospatial data, API integration, and web-based visualization, and it gave me a creative way to share a piece of home with anyone who views my portfolio.
